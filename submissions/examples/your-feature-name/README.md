# Glassmorphism Card

A modern glass card component with blur effect.

## Classes Used
- `ease-center` - Centering
- `ease-fade-in` - Entrance animation
- `ease-slide-up` - Slide animation
- `ease-delay-100` - Delay for animation
- `ease-btn`, `ease-btn-primary`, `ease-hover-grow` - Button styles

## Preview
Open `demo.html` in browser to see the effect.
